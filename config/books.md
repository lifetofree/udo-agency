# 🦆 Duck Book Library

## 🧠 1. Selection Logic (การคัดเลือกหนังสือ)

Udo จะเลือกหนังสือให้ตามระดับพลังงาน (Energy Phase) ดังนี้:

- **🟢 G.1 (แบต 5%):** เหมาะสำหรับช่วง Phase 1 (Ignition) หรือวันที่มีอาการ LOW BAT
  - *ลักษณะ:* มีภาพเยอะ บทสั้นมาก ไม่ต้องใช้ Logic สูง
- **🟡 G.2 (แบต 15%):** เหมาะสำหรับช่วง Phase 5 (Final Push)
  - *ลักษณะ:* บทความสั้น จบในตัว จบแล้วได้ Dopamine ทันที
- **🟠 G.3 (แบต 30%):** เหมาะสำหรับช่วง Phase 4 (Execution)
  - *ลักษณะ:* เนื้อหาไหลลื่น (Flow) มีการเล่าเรื่องที่น่าติดตาม
- **🔴 Boss (แบต 50%+):** เหมาะสำหรับช่วง Phase 2 (Golden Window) เท่านั้น
  - *ลักษณะ:* ศัพท์ยาก วิชาการสูง ต้องใช้สมาธิสูงสุด

## 📜 2. Reading Rules (กฎการอ่านแบบ Duck OS)

- **MVS Strategy:** โหลด **Sample** มาลองก่อน ถ้าไม่สนุก/ยากไปให้ **"ตัดทิ้ง"** ทันที (ห้ามฝืนอ่าน)
